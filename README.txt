Reliant Air Charter OpsSpecs site

Files in this folder:
- index.html
- opspecs.pdf

GitHub Pages setup:
1. Create a new GitHub repository.
2. Upload index.html and opspecs.pdf to the repository root.
3. In GitHub, open Settings > Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. Wait for GitHub Pages to publish your site.
7. Your site URL will look like:
   https://YOUR-USERNAME.github.io/YOUR-REPO/

Direct PDF URL:
https://YOUR-USERNAME.github.io/YOUR-REPO/opspecs.pdf

Specific-page example:
https://YOUR-USERNAME.github.io/YOUR-REPO/opspecs.pdf#page=37

Important:
- GitHub Pages is public by default.
- To update the OpsSpecs later, replace opspecs.pdf with the new file using the same filename.
- Once you have the final PDF URL, I can update the hidden hyperlink inside your GOM.
